Problem:	
	Given an integer n, and n space-separated integers as input. 
	_Create a tuple t, of those n integers
	_Compute and print the result of hash(t) 
	(hash() is a built-in function)

Input Format:
	_The first line contains an integer n ,denoting the number of elements in the tuple
	_The second line contains n space-separated integers describing the elements in tuple 
Output Format:
	Print the result of hash(t)


Sample Input:
	2
	1 2

Sample Output:
	3713081631934410656
